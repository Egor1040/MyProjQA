#01
Авторизація на сайті
1.Step: Зайти на сайт https://automoto.ua/uk/. Expected Result: Сайт відкрився успішно 2.Step: Знайти іконку юзера у верхньому полі сайту та натиснути. Expected Result: Відкрилось модальне вікно для реєстрації або входу в обліковий запис через введення номера телефону. 3.Step: Ввести валідний номер телефону,та натиснути “Продовжити” Expected Result: Відкриється нижче вікно для вводу коду з СМС. 4.Step: Ввести код із СМС та натиснути “Продовжити” Expected Result: Відкриєтьс особистий кабінет з пунктами “Мої оголошення”, “Мої платежі”, “Блокнот”, “Повідомлення” та “Налаштування”.
