Test Cases

#01
Авторизація на сайті
1.Step: Зайти на сайт https://automoto.ua/uk/. Expected Result: Сайт відкрився успішно 2.Step: Знайти іконку юзера у верхньому полі сайту та натиснути. Expected Result: Відкрилось модальне вікно для реєстрації або входу в обліковий запис через введення номера телефону. 3.Step: Ввести валідний номер телефону,та натиснути “Продовжити” Expected Result: Відкриється нижче вікно для вводу коду з СМС. 4.Step: Ввести код із СМС та натиснути “Продовжити” Expected Result: Відкриєтьс особистий кабінет з пунктами “Мої оголошення”, “Мої платежі”, “Блокнот”, “Повідомлення” та “Налаштування”.
#02
Пошук на сайті
1.Step: Зайти на сайт https://automoto.ua/uk/. Expected Result: Сайт відкрився успішно 2.Step: Знайти табло пошуку нижче від основного меню,та обрати пункти і заповнити валідними даними для пошуку (Вибрати марку автомобіля, модель, рік випуску та ціновий діапазон) Expected Result: Обрані пунки заповнені ,відображено в таблі пошуку. 3.Step: Натиснути кнопку “Пошук” Expected Result: Відображено сторінку з результатами обраних параметрів. 4.Step: Перевірити результати пошуку. Expected Result: Список автомобілів відповідає обраним параметрам. Кожен елемент відображає марку, модель, рік випуску і ціну автомобіля.
#03
Перегляд сторінки оголошення про автомобіль
1.Step: Зайти на сайт https://automoto.ua/uk/. Expected Result: Сайт відкрився успішно 2.Step: Знайти табло пошуку нижче від основного меню,та обрати пункти і заповнити валідними даними для пошуку (Вибрати марку автомобіля, модель, рік випуску та ціновий діапазон) Expected Result: Обрані пунки заповнені,відображено в таблі пошуку. 3.Step: Натиснути кнопку “Пошук” Expected Result: Відображено сторінку з результатами обраних параметрів. 4.Step: Натиснути на заголовок оголошення, щоб переглянути деталі автомобіля. Expected Result: Відображається сторінка оголошення про обраний автомобіль з повною інформацією, фотографіями та контактною інформацією продавця.
